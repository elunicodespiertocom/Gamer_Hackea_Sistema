# Gamer_Hackea_Sistema
Hackea el Sistema es un mini-juego web interactivo inspirado en una terminal de hacking. El jugador debe responder correctamente a diferentes retos de informática y ciberseguridad antes de que el sistema lo expulse.  El objetivo es simular una intrusión ficticia estilo laboratorio, resolviendo comandos, conceptos y puertos conocidos.
