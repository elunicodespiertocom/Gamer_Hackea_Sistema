# 🟢 Hackea el Sistema

### Mini Hacker Terminal Game (HTML + JavaScript)

![HTML5](https://img.shields.io/badge/HTML5-Game-orange)
![JavaScript](https://img.shields.io/badge/JavaScript-Vanilla-yellow)
![Cybersecurity](https://img.shields.io/badge/Cybersecurity-Learning-green)
![License](https://img.shields.io/badge/License-MIT-blue)

**Hackea el Sistema** es un mini juego web inspirado en una terminal hacker donde el jugador debe superar retos básicos de informática y ciberseguridad antes de que el sistema lo expulse.

El juego simula un entorno de **penetration testing gamificado**, resolviendo comandos Linux, conceptos de seguridad y preguntas técnicas.

Perfecto para:

* webs personales de tecnología
* demos educativas
* portfolio de ciberseguridad
* contenido interactivo

---

# 🎮 Demo del Juego

El jugador interactúa con una **terminal simulada**.

Debe responder correctamente a preguntas técnicas como:

* comandos Linux
* puertos conocidos
* conceptos de seguridad
* fundamentos de redes

Si falla demasiadas veces o se acaba el tiempo:

🚫 **El sistema bloquea el acceso**

Si completa todos los niveles:

✅ **Obtiene acceso root**

---

# 🧠 Gameplay

### Objetivo

Completar todos los niveles antes de:

* perder las vidas
* que el tiempo llegue a 0

### Mecánica

| Elemento     | Función                             |
| ------------ | ----------------------------------- |
| ⏱ Tiempo     | límite para completar el ataque     |
| ❤ Vidas      | intentos disponibles                |
| 🧠 Preguntas | retos técnicos                      |
| ⭐ Puntos     | recompensa por respuestas correctas |

---

# 🕹 Niveles actuales

| Nivel | Desafío                                  |
| ----- | ---------------------------------------- |
| 1     | Comando Linux para ver directorio actual |
| 2     | Puerto estándar de SSH                   |
| 3     | Comando para listar archivos             |
| 4     | Tipo de contraseña más segura            |
| 5     | Significado de VPN                       |

# 📦 Instalación

Clona el repositorio:

```bash
git clone https://github.com/tuusuario/hackea-el-sistema.git
```

Entra en el directorio:

```bash
cd hackea-el-sistema
```

Abre el archivo:

```
hackea-el-sistema.html
```

en cualquier navegador.
